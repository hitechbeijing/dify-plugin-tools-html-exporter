# html_exporter
由bowenliang123的md_exporter实现了markdown转docx，我在他的基础上重写方法，实现html转docx

<h3>支持以下HTML标签的转换</h3>

```
<p></p>
<div></div>
<span></span>
<ul></ul>
<li></li>
<table></table>
<br\>
<h1></h1>
<h2></h2>
<h3></h3>
<h4></h4>
<h5></h6>
<h6></h6>
<span></span>
<b></b>
<strong></strong>
<i></i>
<u></u>
<em></em>
<u></u>
<small></small>
<ol></ol>
```
<h3>支持css样式：</h3>

```
front-size
front-color#RGB、16进制、3位16进制缩写以及英文常见颜色
font-weight
text_align
margin
line-height
text-decoration
```
